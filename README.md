# Career Tracker

Personal job tracking app for SDE and Product roles.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Import Your Data

1. Go to SDE or Product tracker
2. Click "Import CSV"
3. Select `seed-data/sde-jobs.csv` or `seed-data/product-jobs.csv`

## Features

- ✅ Track SDE and Product job applications
- ✅ Filter by category, status, priority
- ✅ Search across all fields
- ✅ Import/Export CSV
- ✅ Full CRUD operations
- ✅ Dark mode with beautiful UI
- ✅ All data stored locally in browser

## Data Storage

All data is stored in browser localStorage. Export your data regularly as backup!
